▸ COMPONENTES CREADOS
═══════════════════════════════════════════════════════════

1. StatusBadge  (src/components/ui/status-badge.tsx)
   - Badge reutilizable con colores semánticos por entidad/estado
   - Usa statusConfig centralizado + i18n automática via useTranslations()
   - Props: entity (StatusEntity), status (string), className?
   - Entidades soportadas: contracts, projects, proposals, payments

2. EmptyState  (src/components/ui/empty-state.tsx)
   - Componente vacío centrado con ícono, título, descripción y acción opcional
   - Props: icon (LucideIcon), title, description?, action? ({label, href?, onClick?})
   - Reemplaza 8+ patrones inline en todo el dashboard

3. SkeletonLoader  (src/components/ui/skeleton-loader.tsx)
   - 5 variantes exportadas:
     · Pulse (animación base)
     · CardSkeleton (tarjeta con barras pulse)
     · PageSkeleton (header + tabs + N cards)
     · TableSkeleton (header row + N data rows)
     · StatsSkeleton (grid de tarjetas estadísticas)
   - Reemplaza todos los Loader2 spinners de page-level loading

4. AlertDialog  (src/components/ui/alert-dialog.tsx)
   - Componente shadcn basado en @radix-ui/react-alert-dialog
   - Incluye: Dialog, Overlay, Content, Header, Footer, Title,
     Description, Action, Cancel, Trigger, Portal
   - Reemplaza window.confirm() en acciones destructivas

5. Sonner Toaster  (src/app/providers.tsx)
   - Toaster configurado: position="bottom-right", richColors, closeButton
   - Wrapper <Providers> en layout del locale
