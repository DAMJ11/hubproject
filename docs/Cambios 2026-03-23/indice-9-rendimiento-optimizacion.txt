9.5 CLASES CSS UTILIZADAS (tailwindcss-animate)
================================================

  animate-in          — activa animación de entrada
  fade-in             — opacidad 0 → 1
  zoom-in-95          — scale 0.95 → 1
  slide-in-from-top   — translateY(-100%) → 0
  slide-in-from-top-2 — translateY(-0.5rem) → 0
  slide-in-from-bottom-2 — translateY(0.5rem) → 0
  slide-in-from-bottom-4 — translateY(1rem) → 0
  slide-in-from-bottom-6 — translateY(1.5rem) → 0
  slide-in-from-left-4  — translateX(-1rem) → 0
  slide-in-from-right-4 — translateX(1rem) → 0
  duration-200/300/400/500 — duración de animación
  fill-mode-both      — mantiene estado final de animación
  transition-transform — habilita transiciones CSS en transform
  hover:scale-105     — escala al 105% en hover
  active:scale-95     — escala al 95% al presionar
